メタデータ

メタデータとは？

データを説明するデータ


<base>タグはウェブページの基本URLを指定します。

コードを簡潔にしま。

例：<base>タグなし

<a href="http://www.site.com/score/math.html">

<a href="http://www.site.com/score/science.html">

<base>タグあり

...

<a href="math.html">

<a href="science.html">


<link>タグは外部の資源に連列します。

主にスタイルシートを引き起こす際に使います。

<head>

<link type="text.css" rel="stylesheet" href="mystyle.css">

</head>


<meta>タグはウェブページのいろんなメタデータを表現します。

<meta name="author" content="宮澤賢治">

<meta name="discription" content="宮澤賢治の著書についた説明">

<meta name="keyword" content="銀河鉄道の夜、雨ニモマケズ、眼にて云う">

<meta charset="UTF-8">
