video tag

  • src : 경로 지정
  • poster : 비디오가 준비 중일 때의 이미지파일 경로
  • preload : 비디오 재생 전 모두 불러올지 지정
  • autoplay : 자동 재생
  • loop : 반복
  • controls : 재생도구 출력
  • width : 너비
  • height : 높이
