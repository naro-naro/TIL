background 속성

  1. backgroung-image : 배경 이미지 설정
    • 중첩 가능 - 왼쪽에 작성한 이미지가 위로
  2. backgroung-size : 배경이미지 크기 조정 (너비 높이)
    • 100% : 화면에 맞추기
    • 띄어쓰기로 구분 : 가로 세로
    • 콤마로 구분 : 각 이미지에 다른 사이즈 적용
    • contain : 너비 100%
    • cover : 높이 100%
  3. background-repeat : 배경 이미지 바둑판
    • repaet-x : x축 방향을 반복
    • repeat-y : y축 방향으로 반복
    • no-repeat : 반복 안함
  4. backgroung-attachment : 배경이미지를 어떤 방식으로 화면에 붙일지
    1. scroll : 스크롤을 내리면 이미지는 보이지 않게 됨
    2. fixed : 스크롤을 내려도 이미지가 그 자리에 고정되어 있음
  5. background-position : 배경이미지의 위치
    1. background-position : 키워드; (top, bottom, left, right, center, ...)
    2. background-position : x축위치;
    3. background-position : x축위치 y축위치; (0px 50% -> 50% 높이에 이미지 고정)
