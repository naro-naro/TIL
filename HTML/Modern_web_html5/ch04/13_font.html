font 속성

  • font-size : 크기단위 또는 키워드
    • px, em, %
    • large, medium, small, ...
    • cf. h1태그 : 32px / p태그 : 16px
  • font-family : 폰트 지정
    • 폰트 이름이 한 단어 : 따옴표X
    • 폰트 이름이 여러 단어 : 따옴표O
    • 원하는 폰트가 없을 때를 대비하여 여러 폰트를 적기
    • 기본 폰트 : Serif(명조), Sans-serif(고딕), Mono space(고정폭 글꼴)
  • font-style : 폰트 기울기(italic)
  • font-weight : 폰트 두께
    • 기본 두께 : 400
    • 두껍게 : 700 or bold
    • 두께를 지원하지 않는 폰트에는 적용 불가

* text-align : 글자 정렬
left, center, right, ...
inline 형식 태그에는 text-align 사용 불가
(중앙이란 개념이 없음)

  • text-decoration : a태그의 밑줄 제거
