position : absolute

뒤에 입력한 태그가 상위에 올라감

position : absolute의 문제점

  • 1. div 태그가 영역을 차지하지 않는다.
  • 2. 색상 상자가 자신의 부모를 기준으로 위치를 잡지 않는다.

해결법

  • 자손의 position 속성에 absolute를 적용하면 부모는 height 속성 사용하기.
  • 부모의 position 속성에 relative 적용하기.

overflow 속성

    내부 요소가 부모의 범위를 벗어날 때 처리 방법
  • hidden : 영역을 벗어나는 부분을 보이지 않게 처리
  • scroll : 영역을 벗어나는 부분을 스크롤(가로 세로 모두)로 처리
    1. overflow-x : 가로 스크롤만 생성
    2. overflow-y : 세로 스크롤만 생성
